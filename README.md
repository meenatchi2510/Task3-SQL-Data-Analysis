
# Task 3 - SQL for Data Analysis

## Objective
Perform SQL data analysis on an Ecommerce database using SQLite.

## Tools Used
- SQLiteOnline.com
- GitHub

## Dataset
A sample Ecommerce dataset containing:
- Customers
- Products
- Orders
- Order Items

## SQL Concepts Covered
- CREATE TABLE
- INSERT INTO
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- INNER JOIN
- LEFT JOIN
- Multi Table JOIN
- Subqueries
- Aggregate Functions (SUM, AVG)
- Views
- NULL Handling
- Indexes
- Average Revenue Per User (ARPU)

## Repository Contents
- `task3.sql` – Complete SQL script
- `screenshots/` – Output screenshots
- `README.md` – Project documentation

## Outcome
This project demonstrates SQL data analysis techniques by querying and analyzing an Ecommerce database using SQLite.
